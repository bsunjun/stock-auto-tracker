# 공식 리스크 플래그 분류 (Official Risk Flag Taxonomy, KR)

본 문서는 한국거래소(KRX) / KIND / DART 등 공식 시장 채널에서
지정되는 리스크 플래그를 PBKR v4 관점에서 **세 가지 분류**로 나눠
정리한 운영 doctrine 이다. 어떤 분류도 그 자체로 매매 신호가 아니다.
어떤 분류도 자동 실행을 정당화하지 않는다.

본 문서는 실제 종목명, 실제 종목코드, 실제 가격, 실제 공시번호,
실제 주문/계좌 정보를 포함하지 않는다.

## 분류 A — Fundamental / Corporate Hard Exclude

기업의 펀더멘털 / 거버넌스 / 회계 / 재무 자체에 의문이 제기되는
공식 상태이다. **무조건 hard exclude** 다.

대표 카테고리:

- 관리종목 지정.
- 상장폐지 사유 발생 / 상장폐지 실질심사 / 상장적격성 실질심사.
- 거래정지 (회계 이슈, 횡령/배임 혐의, 기간 중 정정 등).
- 불성실공시법인 지정 / 공시번복 / 공시변경 누적.
- 감사의견 비적정 (한정, 부적정, 의견거절).
- 기업의 횡령/배임/회계 처리 위반 정황 공식화.

운영 원칙:

- `RISK_FLAG_PULLBACK_WATCH` 대상이 **아니다**.
- 주도주 후보, 풀백 후보, 관찰 후보 어디에도 포함되지 않는다.
- 일정 기간 내 해소되더라도, 본 doctrine 단독으로 자동 복귀시키지
  않는다.

## 분류 B — Market-Structure Cooling Risk

거래소가 **가격/거래대금/회전율의 과열을 식히려고** 부착하는 경고
유형이다. 기업 펀더멘털에 대한 직접 부정 의견이 아닐 수 있다.

대표 카테고리:

- 투자주의.
- 투자경고 지정예고.
- 투자경고.
- 단기과열종목 지정.

운영 원칙:

- 매매 신호가 **아니다**.
- 일반 `PB_TRIGGER` 로 승격하지 **않는다**.
- 단, 다음 조건이 모두 충족될 때에 한해 별도의 고위험 관찰 상태인
  `RISK_FLAG_PULLBACK_WATCH` 로만 분류한다 (조건은
  `RISK_FLAG_PULLBACK_WATCH_CHECKLIST.md` 참조).
  - 분류 A 가 아님.
  - 주도 업종 / 주도 테마.
  - 실적 / 추정치 상향 또는 구조적 촉매 존재.
  - 첫 눌림.
  - 눌림 거래량 감소.
  - 수급 재점화 신호.
  - 짧고 논리적인 손절 위치.
- 본 분류에 속한 종목을 운영 자동 실행이나 자동 알림 경로에 연결하지
  않는다.

## 분류 C — Extreme Market-Structure Risk

거래소가 가격 폭주 / 정지 임박 / 정지 실행에 한층 더 근접한 단계로
부착하는 경고다. **No entry** 가 원칙이다.

대표 카테고리:

- 투자위험 지정.
- 매매거래정지 임박 / 단계적 정지 적용.
- 투자경고 후 추가 급등으로 정지 조건에 근접한 상태.

운영 원칙:

- 매수 사유가 **아니다**.
- `RISK_FLAG_PULLBACK_WATCH` 대상이 **아니다**.
- 정상화 발표 전까지 신규 진입 / 추가 진입을 하지 않는다.
- 본 분류 종목을 자동 실행이나 자동 알림 경로에 연결하지 않는다.

## 공통 원칙

- 본 분류 자체가 매매 신호가 **아니다**.
- 본 분류는 후보 평가의 사전 필터로 사용된다 — 분류 A 는
  완전 배제, 분류 C 는 신규 진입 배제, 분류 B 는 별도 고위험
  관찰 상태 후보일 수 있다.
- 어떤 분류도 자동 실행, 자동 알림, 본진 진입을 정당화하지 않는다.
- 본 분류 작업 자체는 **read-only** 이고, 결과 기록은 private journal
  에만 둔다. 본 repo 에는 실제 종목 / 가격 / 공시번호 데이터를
  commit 하지 않는다.

## 범위 밖

- 자동 분류 코드.
- 자동 알림 / 자동 주문.
- 실제 종목 raw value, 실제 공시번호, 실제 가격 예시.
- broker / API / Kiwoom / WiseReport / Drive 자동 연결.
