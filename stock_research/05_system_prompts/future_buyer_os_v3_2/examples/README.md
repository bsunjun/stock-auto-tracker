# Examples (synthetic only)

이 폴더의 모든 JSON/문서는 **합성 예시(synthetic)** 입니다.

- 종목명·ticker·숫자 모두 가공된 값입니다.
- 실제 매매 신호가 아닙니다. `direct_trade_signal=false` 유지.
- 실제 PDF 의 추출 텍스트, 실제 거래 데이터, 실제 키움/TradingView 캡처를 **포함하지 않습니다.**
- 학습/검토 목적으로만 사용됩니다.

`synthetic_super_pack_v3_2.example.json` 은 `super_pack_v3_2.schema.json` 의 채움 형태를 보여주기 위한
illustrative 예시입니다.
